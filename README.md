# Panopto Export

Exports videos from a panopto installation in the form of an rss feed.`.
`
If you wish to save the videos themselves, use a tool such as `ffmpeg`. 

## Usage

To use this program you need:

1. The url of the panopto panopto installation installation you want to scrape.
2. Credentials either in the form of a username & password or a valid `.ASPXAUTH` cookie value.
3. Either a folder id or a list of video ids.

See `example-harness.js` for how to call the program.
